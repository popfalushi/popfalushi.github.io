Соревнование: https://www.kaggle.com/competitions/digitaledu123/data?select=train.csv

Привет!
В этом соревновании тебе предстоит построить математическую модель, которая будет предсказывать покупку курса пользователем портала DigitalEdu.

Кейс
В ходе работы над заданием вам необходимо определить пользователей, которые с наибольшей вероятностью купят образовательный продукт на основе информации об аккаунтах пользователей.

Цель
Тебе нужно предсказать, купит ли каждый из пользователей тестового набора данных образовательный курс или нет.

Метрика для оценки результата
Для оценки качества модели мы будем использовать такую метрику, как accuracy score, т.е. процент правильно категоризированных пользователей.

Формат файла с результатами предсказания
В качестве ответа на задание тебе нужно будет загрузить csv-файл, содержащий 1989 строк + строку с заголовками столбцов. Если в твоём файле будут лишние строки или столбцы, платформа выдаст сообщение об ошибке и не примет твоё решение.

Общая информация
Все данные, с которыми ты будешь работать, разделены на два файла:

тренировочный набор (train.csv)
тестовый набор (test.csv)
Тренировочный набор данных содержит различные характеристики каждого пользователя, такие как пол, дата рождения, город и т. д., а также целевую переменную – результат (1, если пользователь купил образовательный продукт и 0 – если нет). Используй тренировочный набор данных для обучения модели. Обрати внимание, что данные нуждаются в очистке и переводе в числовой формат. Для улучшения модели ты также можешь создать новые признаки.

Тестовый набор данных мы будем использовать, чтобы понять, как твоя модель справится с данными, которые она никогда не «видела». В тестовом наборе данных отсутствует столбец с целевой переменной, и твоя задача будет состоять в предсказании этого значения для каждого студента.
